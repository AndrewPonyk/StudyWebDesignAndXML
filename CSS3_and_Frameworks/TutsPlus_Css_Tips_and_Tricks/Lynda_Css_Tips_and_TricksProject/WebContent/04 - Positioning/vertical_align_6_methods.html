Vertical align using display:table-cell

		#child {
	    display: table-cell;
	    vertical-align: middle;
	}

Old and 'good' method for vertical align : position:50% and negative margin

		#oldandgoodParent{
		background: blue;
		width: 400px;
		height: 300px;
		position: relative;
		
	}
	
	#oldandgoodChild{
		width: 200px;
		height: 100px;
		background: yellow;
		position: absolute;
		top: 50%;
		margin-top: -50px;
	}
Negative margin and top 50%

Interesting way to center div

		.div1{
    position:relative;
    width: 300px;
    height: 200px;
    background-color: orange;
}

.div2 {
    height: 100px;
    width:100px;
    background-color: red;
    margin: auto 0;
    position: absolute;
    top: 0; left: 0; bottom: 0; right: 0;
}
Div to be aligned vertically

Also if you want vertical align of single text line - you can use 'line-height' property
