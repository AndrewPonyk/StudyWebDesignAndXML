font-style Значения
normal Обычное начертание текста.
italic Курсивное начертание.
oblique Наклонное начертание. Курсив и наклонный шрифт при всей их похожести не одно и то же. Курсив это специальный шрифт имитирующий рукописный, наклонный же образуется путем наклона обычных знаков вправо.
inherit Наследует значение родителя.
font-weight:
Possible values: bold|bolder|lighter|normal|100|200|300|400|500|600|700|800|900
400 is the same as normal, and 700 is the same as bold
font-variant :
Possible values : inherit, normal, small-caps
Small caps is interesting thing. Modifies all lowercase characters as uppercase reduced size .
